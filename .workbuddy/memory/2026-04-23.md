## Tauri CLI 修复

**问题**：`pnpm tauri dev` 报错 `Cannot find native binding`，找不到 `@tauri-apps/cli-win32-x64-msvc`

**根因**：pnpm 默认使用符号链接（junctions），Windows 上 `@tauri-apps/cli` 无法通过符号链接找到原生模块

**修复方案**：
1. 创建 `.npmrc` 配置文件，设置 `node-linker=hoisted`
2. 删除 node_modules + pnpm-lock.yaml
3. `pnpm install` 重新安装

**文件变更**：
- 新增 `d:\vibe coding\Apartment\A1\apartment-manager\.npmrc`（内容：`node-linker=hoisted`）

**结论**：`pnpm tauri --version` 返回 `tauri-cli 2.10.1`，问题解决
