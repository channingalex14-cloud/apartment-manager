## 3. 待实施变更

### 3a. Schema 新增

```sql
-- monthly_bills 新增字段
ALTER TABLE monthly_bills ADD COLUMN misc_fee INTEGER NOT NULL DEFAULT 0;
ALTER TABLE monthly_bills ADD COLUMN misc_fee_remark TEXT NOT NULL DEFAULT '';
```

### 3b. system_config 新增项

| key | 默认值 | 说明 |
|-----|-------|------|
| 公寓名称 | 新逸公寓 | 通知单标题 |
| 收款户名 | （空） | 银行卡户名 |
| 收款帐号 | （空） | 银行卡号 |
| 收款开户行 | （空） | 开户行 |
| 缴费提示语 | 请及时缴纳房租水电。逾期收取滞纳金。 | 底部提示 |

### 3c. 待确认问题

- [ ] **物业费**：当前 system_config 中的"物业费"是固定金额。样板中物业费=0。
      方案A：物业费走 misc_fee（灵活，备注填"物业费"）
      方案B：monthly_bills 新增独立 property_fee 字段
      **建议选 A**，一张账单只有一笔 misc_fee，备注说明用途

- [ ] **维修费录入时机**：在生成账单时录入，还是账单生成后追加？
      **建议**：generate_room_bill 增加 misc_fee 参数，默认0

- [ ] **管理费计算基准**：样板中管理费实用=电费实用(429度)。
      当前逻辑已是如此，确认无需修改
