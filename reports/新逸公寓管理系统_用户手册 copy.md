## 1. 系统简介

新逸公寓管理系统是一款面向中小型公寓运营者的租金管理工具，支持：

- **房间管理**：记录房间信息、状态（水电表读数）
- **租客管理**：管理租客信息、联系方式
- **合同管理**：租赁合同的全生命周期管理（签约、入住、退房）
- **账单管理**：自动生成月度账单，支持部分支付
- **缴费管理**：记录各类缴费方式（微信/支付宝/银行卡/现金等）
- **押金台账**：押金收取、退还、抵扣全流程跟踪
- **数据备份**：数据库备份与 JSON 数据导出

## 2. 快速开始

### 2.1 首次使用

**步骤 1：配置单价**

1. 进入 **设置** 页面
2. 在「费用单价」中填写：水费单价、电费单价、物业费单价
3. 在「账单参数」中设置：默认押金、账单到期日、滞纳金比例
4. 点击 **保存单价** 和 **保存参数**

**步骤 2：添加房间**

1. 进入 **房间管理** 页面
2. 点击 **添加房间**，填写楼栋、房号、房型、月租
3. 物业费如统一标准可批量设置

**步骤 3：录入租客（可选）**

1. 进入 **租客管理** 页面
2. 点击 **新增租客**，填写姓名、手机号等信息

**步骤 4：签约入住**

1. 进入 **合同管理** 页面
2. 点击 **新建合同**，选择房间、租客，填写起止日期、租金
3. 点击 **确认签约** 完成入住登记

## 4. 房间管理

### 4.1 查看房间列表

进入 **房间管理** 页面，可看到所有房间的：

| 字段 | 说明 |
|------|------|
| 房号 | 楼栋 + 房间号，如「A栋-101」 |
| 房型 | 单间/一室一厅/两室一厅等 |
| 月租 | 基础租金（元/月） |
| 物业费 | 每月物业费（元/月） |
| 状态 | 出租中 / 空房 / 整租 / 自管 / 员工宿舍 |
| 当前租客 | 正在住的租客姓名 |
| 水表读数 | 当前水电表指针读数 |
| 电表读数 | 当前电表指针读数 |

### 4.2 添加房间

点击 **添加房间**，填写：

| 字段 | 必填 | 说明 |
|------|------|------|
| 楼栋 | 是 | 如「A栋」「1号楼」 |
| 房间号 | 是 | 如「101」「1001」 |
| 楼层 | 是 | 数字，如 1 |
| 房型 | 是 | 从预设列表选择 |
| 月租 | 是 | 基础月租金额（元） |
| 物业费 | 否 | 月物业费，不填则按系统默认值 |
| 水表底数 | 否 | 入住时的水表读数 |
| 电表底数 | 否 | 入住时的电表读数 |

### 4.3 编辑房间

点击房间行的 **编辑** 按钮，修改房间信息后保存。

> 注意：修改月租不影响已有合同的租金。

### 4.4 房间状态说明

| 状态 | 说明 |
|------|------|
| 出租中 | 房间当前有租客入住 |
| 空房 | 房间空闲，可供出租 |
| 整租 | 长租整租给企业客户 |
| 自管 | 业主自住 |
| 员工宿舍 | 作为员工宿舍使用 |

### 4.5 查看房间详情

点击房间行的任意位置，可展开查看：
- 房间基本信息
- 当前租客及联系方式
- 当前合同信息
- 水电表读数详情
- 最近 3 条账单记录

## 6. 合同管理

### 6.1 合同列表

显示所有合同，包含：房间、租客、租期、状态（待生效/生效中/已到期/已退房）。

### 6.2 新建合同

点击 **新建合同**，填写：

| 字段 | 必填 | 说明 |
|------|------|------|
| 房间 | 是 | 从下拉列表选择 |
| 租客 | 是 | 从已有租客选择或新建 |
| 合同编号 | 是 | 建议格式：楼栋-房号-年月，如「A101-202603」 |
| 合同起止日期 | 是 | 租赁期限 |
| 月租金 | 是 | 每月租金金额 |
| 物业费 | 否 | 每月物业费，不填则按系统默认值 |
| 押金 | 是 | 入住押金金额 |
| 付款方式 | 是 | 月付/季付/年付 |
| 入住日期 | 是 | 实际入住的日期 |

### 6.3 合同状态

| 状态 | 说明 |
|------|------|
| 待生效 | 合同尚未开始（起效日期在将来） |
| 生效中 | 合同正在执行 |
| 已到期 | 合同到期但未退房 |
| 已退房 | 已完成退房结算 |

### 6.4 入住登记

合同生效后，点击 **入住登记**，系统将：
- 更新房间状态为「出租中」
- 记录入住时间

### 6.5 退房结算

点击 **退房**，填写退房日期，系统将：
- 更新房间状态为「空房」
- 生成末月账单（水电按实际天数结算）
- 更新合同状态为「已退房」

### 6.6 违约记录

对于欠租、损坏财物等违约情况，点击 **违约登记**，系统将记录违约类型和详情。

## 7. 账单管理

### 7.1 账单列表

进入 **账单管理** 页面，可按年月、房间、状态筛选账单。

| 字段 | 说明 |
|------|------|
| 房号 | 楼栋-房间号 |
| 租客 | 当前租客姓名 |
| 年月 | 账单所属月份 |
| 应收金额 | 应缴总额 |
| 实缴金额 | 已缴金额 |
| 状态 | 待缴费 / 已支付 / 部分支付 / 已作废 |

### 7.2 生成月度账单

点击 **生成账单**：

1. 选择年月（默认当前年月）
2. 可选：只生成特定房间的账单
3. 可选：设置额外费用（如维修费）
4. 点击 **开始生成**

> 生成规则：
> - 按抄表读数计算水电费
> - 租金按月计算（首月/末月按实际入住/退房天数比例计算）
> - 账单自动关联对应合同和租客

### 7.3 查看账单详情

点击账单行，展开详情，包含：

**费用明细：**
- 租金（本月天数 × 日租金）
- 物业费
- 水费：（本次读数 - 上次读数）× 单价
- 电费：（本次读数 - 上次读数）× 单价
- 上月结转
- 其他费用（维修费等）
- 合计金额

**操作权限：**
- **确认支付**：标记为全额支付
- **部分支付**：录入实缴金额（适用于提前部分缴费）
- **作废**：作废该账单（需谨慎操作）

### 7.4 打印收费通知单

在账单详情页，点击 **打印**，可预览并打印收费通知单。

收费通知单包含：
- 公寓名称、地址、银行账户信息
- 房号、租客姓名、年月
- 费用明细表格
- 收款二维码

### 7.5 归档历史账单

对于超过 12 个月的账单，建议归档：

1. 在 **设置 → 归档历史账单** 中选择年月
2. 点击 **归档选中月份**
3. 归档后账单默认不显示，数据不会丢失

## 8. 缴费管理

### 8.1 缴费记录列表

显示所有缴费记录，可按年月、房间、缴费方式筛选。

| 字段 | 说明 |
|------|------|
| 房号 | 缴费房间 |
| 金额 | 缴纳金额（分） |
| 缴费日期 | 实际缴费日期 |
| 缴费方式 | 微信/支付宝/银行卡/现金/商家码/押金抵扣/混合支付 |
| 关联账单 | 所缴纳的账单年月 |

### 8.2 记录缴费

点击 **记录缴费**：

1. 选择房间（必填）
2. 输入缴费金额（必填）
3. 选择缴费日期（默认当天）
4. 选择缴费方式（必填）
5. 如需关联账单，选择年月
6. 点击 **确认缴费**

### 8.3 作废缴费记录

如缴费记录有误，点击 **作废**，系统将：
- 标记该记录为已作废
- 如关联了账单，自动回滚账单实缴金额

## 9. 押金台账

### 9.1 押金台账页面

记录所有押金的收支流水，包括：

- 押金收取（入住时）
- 押金退还（退房时）
- 押金抵扣（自动抵扣欠款）
- 押金转结（换房时转入新房）

### 9.2 押金记录说明

| 类型 | 说明 |
|------|------|
| 收取 | 入住时收取的押金 |
| 退还 | 退房时退还的押金 |
| 抵扣 | 用于抵扣欠租、水电费等 |
| 转结 | 换房时押金转入新房 |

押金余额 = 历史收取 - 历史退还 - 历史抵扣

## 10. 抄表管理

### 10.1 为什么要抄表

每月生成账单前，需要先记录水电表读数。系统根据：
**费用 =（本次读数 - 上次读数）× 单价**
自动计算水电费。

### 10.2 记录单条读数

1. 进入 **房间管理**，找到目标房间
2. 点击房间行的 **编辑**
3. 在「水表底数」「电表底数」中填入当前读数
4. 点击 **保存**

### 10.3 批量记录读数

1. 进入 **设置 → 数据维护**
2. 找到 **批量抄表** 功能
3. 填写年月
4. 逐行输入各房间的水电表读数
5. 点击 **保存**

> 提示：建议每月固定一天（如每月 1 日）抄表，便于前后对比。

## 11. 数据备份与导出

### 11.1 手动备份

1. 进入 **设置 → 数据备份**
2. 点击 **立即备份**
3. 系统将执行 VACUUM 并复制数据库文件到备份目录

### 11.2 自动备份

1. 在 **数据备份** 卡片中
2. 开启 **启用自动备份**
3. 设置保留份数（默认 7 份）
4. 点击 **保存设置**

启用后，系统每日自动备份一次，并自动清理超过保留份数的旧备份。

### 11.3 从备份恢复

1. 在 **备份记录** 表格中找到目标备份
2. 点击该行的 **恢复**
3. 确认警告信息，点击确定

> 警告：恢复操作将覆盖当前数据库，现有数据将丢失！

恢复前系统会自动创建紧急备份（`apartment.db.emergency`）。

### 11.4 数据导出

1. 进入 **设置 → 数据导出**
2. 选择导出类型：
   - **房间数据**：所有房间基本信息
   - **租客数据**：所有租客联系方式
   - **账单数据**：所有账单明细（支持年月筛选）
   - **缴费记录**：所有缴费记录（支持年月筛选）
   - **年度汇总**：各月账单统计
3. 如需筛选年月，选择年月
4. 点击 **导出 JSON**
5. 选择保存位置

导出的 JSON 为 UTF-8 编码，可用 Excel 或文本编辑器打开。

## 12. 系统设置

### 12.1 费用单价

| 项目 | 说明 |
|------|------|
| 水费单价 | 元/吨 |
| 电费单价 | 元/度 |
| 物业费单价 | 元/月 |

> 修改单价不影响已生成的账单。

### 12.2 账单参数

| 项目 | 说明 |
|------|------|
| 默认押金 | 新建合同时的默认押金金额 |
| 账单到期日 | 每月几号为到期日（影响催缴提醒） |
| 滞纳金比例 | 逾期每天按此比例收取滞纳金 |

### 12.3 收款账户信息

设置收费通知单上显示的收款信息：
- 公寓名称、楼栋地址
- 银行户名、银行帐号、开户行
- 温馨提示（如"请及时缴纳房租水电"）
- 收款二维码（将图片放入 `public/bill-assets/` 目录）

### 12.4 数据导入

可将 Excel 格式的历史账单数据导入系统。

**格式要求：** Excel 文件需包含指定列（参考系统提供的模板）。

**导入流程：**
1. 点击 **选择文件** → 选择 Excel 文件
2. 选择目标年月
3. 点击 **开始导入**
4. 查看导入结果（成功条数、跳过条数、错误列表）

> 注意：已存在的账单不会重复创建，导入前可点击 **诊断** 检查文件格式。

## 13. 常见问题

### Q1：生成账单时房间没有租客怎么办？

检查该房间是否有生效中的合同。进入 **合同管理**，确认合同状态为「生效中」且入住日期已到。

### Q2：账单金额与预期不符？

1. 检查抄表读数是否正确（本次读数应 ≥ 上次读数）
2. 检查单价设置是否正确（**设置 → 费用单价**）
3. 检查账单类型是否正确（如首月免水电、月中入住等特殊计算）

### Q3：租客提前退房，账单如何结算？

1. 在 **合同管理** 中点击该合同的 **退房**
2. 填写退房日期，系统自动生成末月账单
3. 末月租金按实际入住天数计算

### Q4：押金忘记退了怎么办？

进入 **押金台账**，手动添加一条「退还」记录即可。

### Q5：如何更改房间的水电表读数？

进入 **房间管理**，点击房间的 **编辑**，修改水表底数/电表底数。注意：修改后建议重新生成对应月账单。

### Q6：误操作作废了账单，可以恢复吗？

作废只是标记，不删除数据。联系技术支持，通过数据库恢复。

### Q7：备份恢复后数据不对？

恢复前系统会自动创建紧急备份 `apartment.db.emergency`。如恢复出现问题，用紧急备份重新恢复即可。

### Q8：导出的 JSON 文件打开乱码？

使用支持 UTF-8 的编辑器（如 VS Code、Notepad++）。Windows 记事本建议不要直接打开。

### Q9：如何添加新的房间类型？

目前房间类型为预设列表，修改需要联系技术支持或直接在数据库 `system_config` 表中配置。

### Q10：系统数据存在哪里？

数据库文件位于应用数据目录下：
```
{应用数据目录}/apartment.db
```

## 附录

### A. 操作日志说明

所有关键操作都会记录到操作日志中，包括：
- 合同创建/激活/退房
- 入住/退房登记
- 账单生成/支付/作废
- 缴费记录增删改

日志记录操作人、操作类型、操作时间和影响范围，便于审计和排查问题。

### B. 数据安全建议

1. **定期备份**：建议每月手动备份一次，重要操作前先备份
2. **启用自动备份**：开启后系统每日自动备份
3. **保留多份备份**：保留至少 3 份以上不同时间点的备份
4. **导出重要数据**：定期导出账单和缴费记录到 Excel
