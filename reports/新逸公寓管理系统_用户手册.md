## 1. 系统概述

### 1.1 导航结构

系统左侧为可折叠侧边栏，包含以下菜单：

| 菜单项 | 路径 | 说明 |
|--------|------|------|
| 首页概览 | `/dashboard` | 运营数据仪表盘 |
| 房态管理 | `/rooms` | 房间卡片视图，含详情抽屉 |
| 账单管理 | `/bills` | 账单列表与生成 |
| 缴费管理 | `/payments` | 缴费记录列表 |
| 系统设置 | `/settings` | 系统配置与数据管理 |
| 提醒管理 | `/reminders` | 事务提醒管理 |
| 租客管理 | `/tenants` | 租客信息管理 |

> **注意**：合同管理、押金台账无独立菜单入口，均通过房态管理或账单页面的抽屉/dialog 操作。

### 1.2 全局操作栏

顶部 Header 包含：

- **折叠按钮**：收起/展开左侧菜单
- **面包屑**：显示当前页面路径
- **全局搜索**：输入房号或租客姓名，下钻到对应房间
- **刷新按钮**：刷新当前页面数据
- **全局月选择器**：切换查看的年月（影响账单筛选）
- **主题切换**：深色/浅色模式

## 2. 快速开始

### 步骤 1：登录系统

1. 启动应用后，输入账号和密码
2. 默认管理员账号：`admin` / `admin123`
3. 登录成功后进入首页仪表盘

> **注意**：系统启动时会自动跳转到登录页面，未登录用户无法访问任何功能。

### 步骤 2：配置单价

1. 进入 **系统设置**
2. 在「费用单价」中填写：水费单价（元/吨）、电费单价（元/度）、物业费单价（元/月）
3. 在「账单参数」中设置：默认押金、账单到期日、滞纳金比例
4. 点击 **保存参数**

### 步骤 3：添加房间

1. 进入 **房态管理**
2. 点击左上角 **添加房间**
3. 填写楼栋、房间号、楼层、房型、月租
4. 押金和物业费可单独编辑

### 步骤 4：录入租客

1. 进入 **房态管理**，点击某房间卡片
2. 在右侧抽屉中，点击 **费用录入** 或 **修改状态** 旁的入口
3. 或进入 **系统设置 → 导入账单** 批量导入

> 租客信息通常在新建合同或导入账单时一并创建。

### 步骤 5：签约入住

1. 进入 **房态管理**，点击目标房间卡片
2. 在右侧抽屉中找到 **合同** 相关操作
3. 新建合同后，点击 **入住登记**

## 4. 房态管理

路径：`/rooms`

### 4.1 视图说明

房态管理采用**卡片网格视图**，每张卡片代表一个房间。

卡片正面显示：房号（点击可打印收费通知单）、状态标签、房型、当前租客、联系电话、月租。

### 4.2 房间状态（8种）

| 状态 | 颜色 | 说明 |
|------|------|------|
| 在租 | 绿色 | 正常出租中 |
| 新租 | 绿色 | 新签约入住 |
| 空房 | 黄色 | 房间空闲 |
| 员工 | 绿色 | 员工宿舍 |
| 管理 | 绿色 | 自管房间 |
| 违约 | 红色 | 欠租等违约情况 |
| 待清洁 | 紫色 | 退房后待清洁 |
| 维修中 | 橙色 | 正在维修 |

### 4.3 房间卡片详情

点击任意房间卡片，右侧滑出详情抽屉（宽 400px），包含：

**基本信息区**
- 房号 + 状态标签
- 房型 + 楼层

**费用概览**
- 押金（元）
- 月租（元）
- 物业费（元/月）

**租客信息**
- 租客姓名
- 联系电话

**合同信息**
- 合同开始日期
- 合同结束日期
- 到期提醒（安全/即将到期/已到期）

**水电表读数**
- 水表：上期读数 → 本期读数 = 用量（吨）
- 电表：上期读数 → 本期读数 = 用量（度）

**近期账单**
- 最近几条账单记录：年月、总额、状态

### 4.4 卡片操作

详情抽屉底部提供操作按钮：

| 按钮 | 功能 |
|------|------|
| 编辑押金 | 修改房间押金金额 |
| 费用录入 | 补充其他费用 |
| 修改状态 | 变更房间状态（需二次确认） |
| 📍 图标 | 打印该房间的收费通知单 |

### 4.5 全局搜索

顶部搜索框支持按**房号**或**租客姓名**精确过滤。

## 6. 生成月度账单

路径：`/bills/generate`

### 6.1 操作步骤

1. 选择年月（默认当前月）
2. 系统显示所有可生成账单的房间列表（空房和在租状态）
3. 可逐行修改各房间的本次水表/电表读数
4. 点击 **重置读数** 可恢复为房间当前的底数
5. 点击 **生成账单**

> 注意：未填写水电表读数的房间会被跳过。

### 6.2 生成规则

- 水电费 =（本次读数 - 上次读数）× 单价
- 首月/末月按实际入住/退房天数比例计算租金
- 账单自动关联对应合同和租客

## 7. 账单详情与操作

路径：`/bills/:id`

### 7.1 费用明细

账单详情页展示完整分项：

| 项目 | 说明 |
|------|------|
| 租金 | 月租或按天比例 |
| 物业费 | 月物业费 |
| 水费 | 包含用量（吨） |
| 电费 | 包含用量（度） |
| 管理费 | 月管理费 |
| 维修费 | 如有 |
| 杂费 | 其他费用 |
| 上期余额 | 上月结转欠款 |
| 合计 | 应收总额 |
| 已付 | 已缴金额 |
| 剩余 | 待缴金额 |

### 7.2 操作按钮

根据账单状态，显示不同操作：

| 操作 | 条件 | 说明 |
|------|------|------|
| 打印通知单 | 始终显示 | 跳转打印页面 |
| 确认支付 | 可确认时 | 标记为全额支付 |
| 部分支付 | 可部分支付时 | 输入实缴金额 |
| 作废账单 | 可作废时 | 作废该账单 |

### 7.3 打印收费通知单

点击 **打印通知单** 跳转 `/bills/:id/print`，包含：
- 公寓名称、地址、银行账户
- 房号、租客姓名、账单年月
- 费用明细表格
- 收款二维码

支持直接浏览器打印或另存 PDF。

## 9. 系统设置

路径：`/settings`

共 9 个功能区块：

### 9.1 费用单价

| 字段 | 说明 |
|------|------|
| 水费单价 | 元/吨 |
| 电费单价 | 元/度 |
| 物业费单价 | 元/月 |

> 修改不影响已生成账单。

### 9.2 账单参数

| 字段 | 说明 |
|------|------|
| 默认押金 | 新建合同时的默认押金金额 |
| 账单到期日 | 每月几号为到期日 |
| 滞纳金比例 | 逾期每天按此比例收取滞纳金 |

### 9.3 收款账户信息

设置收费通知单显示的收款信息：
- 公寓名称、楼栋地址、银行户名、银行帐号、开户行
- 温馨提示
- 收款二维码（将图片放入 `public/bill-assets/` 目录）

### 9.4 全部配置项

只读表格，显示所有系统配置项（键/值/类型/描述/状态）。

### 9.5 数据维护

**执行数据库压缩**：清理删除记录后未释放的空间，建议每月执行一次。

### 9.6 导入账单

将 Excel 格式的历史账单数据导入系统。

1. 点击 **选择文件** → 选 Excel
2. 选择目标年月
3. 点击 **开始导入**
4. 查看导入结果

> 点击 **诊断** 可预先检查文件格式是否正确。

### 9.7 归档历史账单

归档可将历史月份账单标记为"已归档"，默认不显示但数据不丢失。

1. 选择年月
2. 点击 **归档选中月份**
3. 已归档月份可在下方表格中随时恢复

### 9.8 数据备份

**手动备份**：点击 **立即备份**，系统执行 VACUUM 并复制数据库到备份目录。

**自动备份**：
- 开启 **启用自动备份** 开关
- 设置 **保留份数**（默认 7 份）
- 点击 **保存设置**

**备份记录**：显示所有备份文件，支持**恢复**（覆盖当前数据库）和**删除**。

> 恢复操作会覆盖当前数据！恢复前系统自动创建紧急备份。

### 9.9 数据导出

导出数据为 JSON 文件（UTF-8 编码，可用 Excel 打开）：

| 类型 | 说明 |
|------|------|
| 房间数据 | 所有房间基本信息 |
| 租客数据 | 所有租客联系方式 |
| 账单数据 | 所有账单明细（支持年月筛选） |
| 缴费记录 | 所有缴费记录（支持年月筛选） |
| 年度汇总 | 各月账单统计 |

## 10. 常见问题

**Q1：生成账单时房间没有租客？**

检查该房间是否有生效中的合同。确认合同状态为"生效中"且入住日期已到。

**Q2：账单金额与预期不符？**

1. 检查抄表读数是否正确（本次 ≥ 上次）
2. 检查单价设置是否正确（系统设置 → 费用单价）
3. 确认账单类型是否正确（首月免水电、月中入住等特殊计算）

**Q3：租客提前退房，账单如何结算？**

在合同列表中找到该合同，点击 **退房**，填写退房日期，系统自动生成末月账单（按实际入住天数结算）。

**Q4：如何查看押金台账？**

进入 **房态管理**，点击某房间卡片，在详情抽屉的合同信息区可看到押金余额。

**Q5：如何更改房间的水电表读数？**

进入 **房态管理**，点击房间卡片，在详情抽屉中查看当前读数。如需修改，通过"生成账单"流程中的抄表步骤更新。

**Q6：误操作作废了账单，可以恢复吗？**

作废只是标记，不删除数据。联系技术支持通过数据库恢复。

**Q7：备份恢复后数据不对？**

恢复前系统自动创建紧急备份 `apartment.db.emergency`。用紧急备份重新恢复即可。

**Q8：导出的 JSON 文件打开乱码？**

使用支持 UTF-8 的编辑器（VS Code、Notepad++）。避免使用 Windows 记事本。

**Q9：如何添加新的房间类型？**

目前房间类型为预设列表，修改需联系技术支持。

**Q10：提醒功能在哪里？**

提醒功能已在侧边栏加入，路径为 `/reminders`。

**Q11：如何退出登录？**

点击顶部导航栏右侧的用户名，会显示登出按钮，点击即可退出当前账号。

**Q12：忘记了密码怎么办？**

请联系系统管理员重置密码。
