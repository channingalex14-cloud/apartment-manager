## 二、表结构详解

### 2.1 rooms（房间表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| room_number | TEXT UNIQUE | 房号，如 "101", "1203" |
| floor | INTEGER | 楼层 1-14 |
| building | TEXT | 楼栋，默认 "58栋" |
| room_type | TEXT | 房型：单间/一房一厅/二房一厅/三房一厅/商铺 |
| base_rent | INTEGER | 月租金（分） |
| property_fee | INTEGER | 物业费（分/月） |
| deposit | INTEGER | 标准押金（分） |
| status | TEXT | 当前状态 |
| water_meter_current | INTEGER | 当前水表读数 |
| electric_meter_current | INTEGER | 当前电表读数 |
| is_deleted | INTEGER | 软删除标志 |
| created_at | TEXT | 创建时间 |
| updated_at | TEXT | 更新时间 |

**状态枚举**：`空房` `在租` `新租` `员工` `管理` `违约` `维修中` `待清洁`

### 2.2 tenants（租客表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| name | TEXT | 姓名（必填） |
| phone | TEXT | 手机号（必填） |
| phone2 | TEXT | 备用手机号 |
| emergency_contact | TEXT | 紧急联系人 |
| emergency_phone | TEXT | 紧急联系电话 |
| is_deleted | INTEGER | 软删除标志 |

**设计说明**：手机号不设 UNIQUE，因为运营商回收号码后可能重新分配。同一租客通过 tenant_history 追踪。

## 四、软删除策略

所有业务表均包含 `is_deleted` 字段：

```
is_deleted = 0  →  正常数据
is_deleted = 1  →  已删除（逻辑删除）
```

**查询规范**：所有 SELECT 必须加上 `WHERE is_deleted = 0` 过滤条件。

## 五、金额存储规范

所有金额字段以**整数分**存储，前端显示时 /100 转为元。

**示例**：
- `base_rent = 200000` → 表示 2000.00 元
- `total_amount = 258600` → 表示 2586.00 元

**转换函数**（前端）：
```typescript
// 分 → 元（显示）
const toYuan = (fen: number) => (fen / 100).toFixed(2)

// 元 → 分（存储）
const toCent = (yuan: number) => Math.round(yuan * 100)
```

## 七、用户权限表（Phase 4）

### 7.1 users（用户表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| username | TEXT UNIQUE | 用户名（唯一） |
| password_hash | TEXT | bcrypt 密码哈希 |
| role | TEXT | 角色：admin/frontdesk/maintenance/finance |
| display_name | TEXT | 显示名称 |
| is_active | INTEGER | 1=启用 0=禁用 |
| last_login_at | TEXT | 最后登录时间 |
| created_at | TEXT | 创建时间 |
| updated_at | TEXT | 更新时间 |

**默认账号**：`admin` / `admin123`

**角色说明**：

| 角色 | 说明 |
|------|------|
| admin | 管理员，拥有全部权限 |
| frontdesk | 前台，负责日常接待和缴费 |
| maintenance | 维修人员，负责房间维护 |
| finance | 财务，负责账单和财务报表 |

### 7.2 permissions（权限表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| role | TEXT | 角色 |
| permission | TEXT | 权限标识 |
| resource | TEXT | 资源类型 |
| action | TEXT | 操作：read/write/manage/none |
| description | TEXT | 权限描述 |
| created_at | TEXT | 创建时间 |

**UNIQUE约束**：`role + permission + resource` 联合唯一
